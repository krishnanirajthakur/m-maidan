# M-Maidan

Real-time stadium companion app with AI guidance, crowd heatmap, smart paths, and express ordering.

## ✨ Features
- **Live Crowd Heatmap**: Visualizes crowd density across stadium sections/gates
- **Smart Path Recommendations**: Optimal routes based on real-time data
- **Express Order Tracking**: Nachos & Beer status updates
- **Gaj-Guide AI**: Floating elephant mascot with contextual tips (🐘)
- **Dual Themes**: Day (Neumorphic Crystal) / Night (Glassmorphic Space)
- **Offline Mode**: Graceful degradation
- **Full-Screen Stadium Background**

## 🚀 Quick Start

```bash
cd m-maidan
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## 🛠 Tech Stack
- React + Vite
- Tailwind CSS
- Framer Motion (animations)
- Custom Hooks/Context (Theme, Stadium Data)

## 📱 Responsive
Optimized for mobile stadium use. Full-screen background adapts to all devices.

## 🎨 Customization
- Edit `src/assets/background.png` for stadium theme
- Toggle theme via header button
- Mock data in `src/hooks/useStadiumData.js`
